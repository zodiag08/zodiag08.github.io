Introduction à Bootstrap

Bootstrap est un framework open-source populaire pour le développement d'interfaces web réactives et adaptatives. Créé par Mark Otto et Jacob Thornton chez Twitter en 2011, il a été conçu pour simplifier et standardiser la création de sites web modernes. Grâce à sa large adoption, Bootstrap est aujourd'hui l'un des frameworks front-end les plus utilisés.

Versions de Bootstrap

Depuis sa création, Bootstrap a évolué à travers plusieurs versions, chacune apportant des améliorations, des corrections de bugs et de nouvelles fonctionnalités :

Bullstrap 1 (2011)

La première version de Bootstrap se concentrait sur les bases du responsive design et incluait un ensemble de composants prêts à l'emploi (grilles, boutons, formulaires, etc.).

Bullstrap 2 (2012)

Cette version a introduit une grille fluide, un système de mise en page plus flexible. Le framework est devenu plus modulable et prêt à l’emploi pour des designs réactifs sur différentes tailles d'écran.

Bullstrap 3 (2013)

Bootstrap 3 a radicalement changé la donne avec la transition vers un design mobile-first. Cette version a également intégré un design plus "plat", supprimant les bordures et ombres des éléments pour adopter une esthétique plus moderne.

Bullstrap 4 (2018)

Cette version a apporté des changements majeurs avec l’introduction de Flexbox pour la mise en page, des composants plus personnalisables, ainsi qu’un nouveau système de couleurs et des outils pour la gestion des typographies. Elle a également abandonné les dépendances de jQuery et a adopté Sass pour la gestion des styles.

Bullstrap 5 (2021)

La version la plus récente a encore supprimé la dépendance à jQuery, ajouté de nouveaux composants, amélioré les options de personnalisation, et optimisé les performances. Elle a introduit une grille plus flexible, des améliorations pour les modals, et des outils améliorés pour les thèmes. Les icônes Bootstrap ont aussi été séparées en un package distinct.

Avantages de Bootstrap:

Bootstrap offre de nombreux avantages qui en font un choix privilégié pour les développeurs web

Gains de productivité

Avec des composants réutilisables et des classes prêtes à l’emploi, le développement d’interfaces devient beaucoup plus rapide, permettant aux développeurs de se concentrer sur les fonctionnalités et l’optimisation.

Documentation complète et communauté active

La documentation de Bootstrap est extrêmement détaillée et bien structurée, offrant des exemples clairs de son utilisation. De plus, Bootstrap bénéficie d'une communauté large et active qui contribue au développement du framework, créant des outils et des ressources supplémentaires.

Personnalisation facile

Grâce à l'utilisation de Sass (dans les versions 4 et 5), il est possible de personnaliser facilement les couleurs, les espacements et d'autres aspects du design sans modifier le code source de base.

Système de grilles flexible

Le système de grilles de Bootstrap permet de diviser facilement l'écran en différentes colonnes et d'adapter la mise en page en fonction de la taille de l'écran. Cela facilite la création de designs fluides et réactifs.

Système de Grid

Definition

Le système de grille de Bootstrap est un système flexible et réactif basé sur des lignes et des colonnes, qui permet de créer des mises en page adaptatives pour différents types d'écrans (smartphones, tablettes, ordinateurs de bureau). Le système repose sur une grille de 12 colonnes, et la largeur des éléments peut être ajustée en fonction de l'espace disponible.



Le système de grille utilise trois éléments principaux :

Conteneur (container) : C'est l'élément de base qui contient la grille. Il peut être fluide (container-fluid) ou fixe (container). Ligne (row) : Une ligne qui contient plusieurs colonnes.
Colonne (col) : Chaque colonne occupe une certaine largeur dans la ligne, et la largeur est calculée par rapport au nombre de colonnes que vous définissez (sur un total de 12 colonnes).




Maitriser les classes CSS de base

Pour maîtriser les classes CSS de base, notamment celles qui concernent la typographie et les liens, il est essentiel de comprendre les propriétés CSS les plus courantes utilisées pour ces éléments. Voici un aperçu des propriétés de base, accompagnées d'exemples pour vous aider à mieux les comprendre et les utiliser.

1. Typographie (Fonts)

La typographie est essentielle pour le rendu du texte. Vous allez manipuler des éléments comme la taille de la police, le type de police, l'espacement des lignes,
Propriétés de base pour la typographie :
font-family : Définit le type de police.
font-size : Définit la taille de la police.
font-weight : Définit l'épaisseur du texte.
font-style : Permet de mettre le texte en italique.
line-height : Définit l'espacement entre les lignes de texte.
letter-spacing : Définit l'espacement entre les lettres.
text-transform : Permet de mettre le texte en majuscules ou minuscules.
text-align : Définit l'alignement du texte (gauche, droite, centré). br color : Définit la couleur du texte.

2. Liens (Links)

Les liens sont un autre élément crucial pour la navigation dans une page web. Vous pouvez styliser les liens pour améliorer leur apparence et les rendre interactifs.
Propriétés de base pour les liens :
color : Définit la couleur du lien.
text-decoration : Permet de définir ou d'enlever le soulignement.
hover : Pseudo-classe appliquée lorsque le curseur passe sur un lien.
visited : Pseudo-classe appliquée aux liens déjà visités.
active : Pseudo-classe appliquée au lien pendant que l'utilisateur clique dessus.
cursor : Change l'apparence du curseur lorsqu'il survole un lien.

3.Tableaux

Pour maîtriser les classes CSS de base utilisées pour la mise en forme des tableaux,
il est important de comprendre les différentes propriétés qui influencent l'apparence des éléments
table, tr, th, ,ainsi que les bordures, les espacements et l'alignement du contenu.
Voici une introduction détaillée aux propriétés CSS que vous pouvez utiliser pour styliser vos tableaux.
1. Structure de base d'un tableau HTML
Un tableau HTML est composé des éléments suivants : table : Le conteneur principal du tableau.
tr : Définit une ligne dans le tableau.
th : Définit une cellule d'en-tête dans une ligne (généralement en gras et centré par défaut).
td : Définit une cellule de données dans une ligne.
2. Propriétés de base pour styliser un tableau CSS
a. border : Ajouter des bordures
La bordure permet de définir les lignes qui séparent les cellules d'un tableau.
Vous pouvez utiliser la propriété border pour ajouter des bordures aux éléments du tableau,
comme la table elle-même, les cellules de données ou les en-têtes.
b. border-collapse : Fusionner les bordures
collapse : Fusionne les bordures des cellules adjacentes (souvent utilisé pour les tableaux classiques).
separate : Ajoute un espacement entre les cellules, ce qui est utile pour un tableau avec un design aéré.
c. padding : Espacement interne des cellules
L'espacement à l'intérieur des cellules peut être ajusté en utilisant padding.
d. text-align et vertical-align : Alignement du texte
text-align : Aligne le contenu horizontalement dans les cellules (gauche, droite, centre).
vertical-align : Aligne le contenu verticalement dans les cellules (haut, milieu, bas).
e. background-color : Couleur de fond
Vous pouvez changer la couleur de fond de la table, des en-têtes ou des cellules de données.
f. color : Couleur du texte
Permet de définir la couleur du texte dans les cellules.

4. Eléments de formulaires et contrôles supportés

Les éléments de formulaires et contrôles dans Bootstrap sont des composants de formulaire utilisés pour la collecte et l'interaction des utilisateurs avec les données. Voici une définition concise en 10 lignes :
-Champ de texte (input type="text") : Permet à l'utilisateur de saisir du texte.
-Champ email (input type="email") : Spécialement conçu pour la saisie d'adresses email avec validation intégrée.
-Mot de passe (input type="password") : Permet la saisie sécurisée d'un mot de passe.
-Zone de texte (textarea) : Utilisé pour saisir plusieurs lignes de texte.
-Bouton (button ou input type="submit") : Soumet le formulaire.
-Case à cocher (input type="checkbox") : Permet de sélectionner une ou plusieurs options.
-Radio boutons (input type="radio") : Permet de choisir une option parmi plusieurs.
-Liste déroulante (select) : Permet à l'utilisateur de choisir parmi une liste d'options.
-Étiquette (label) : Associe des étiquettes aux champs de formulaire pour l'accessibilité.
-Validations (required, min, max, etc.) : Attributs HTML pour valider les champs du formulaire.
-Ces éléments sont stylisés par Bootstrap avec des classes comme form-control et btn pour une présentation moderne et réactive.

5. Boutons

Les boutons sont des éléments essentiels d'une interface web, car ils permettent aux utilisateurs d'interagir avec votre site. Styliser les boutons avec CSS peut améliorer leur apparence et leur accessibilité, tout en rendant l'interaction plus agréable. Pour maîtriser les classes CSS de base pour les boutons, voici un guide détaillé sur les propriétés et les techniques que vous pouvez utiliser pour personnaliser l'apparence de vos boutons.
1. Structure de base d'un bouton HTML
En HTML, vous pouvez créer un bouton avec l'élément (button). Il existe aussi d'autres éléments comme (input type="button") ou (a) qui peuvent être stylisés comme des boutons.
2. Propriétés CSS essentielles pour les boutons
Les boutons peuvent être stylisés de différentes façons en utilisant les propriétés CSS suivantes :
+a. background-color : Couleur de fond
La couleur de fond du bouton est une des propriétés les plus utilisées. Vous pouvez choisir n'importe quelle couleur en utilisant des noms de couleurs, des valeurs hexadécimales, RGB, ou même HSL.
+b. border : Bordure du bouton
Les boutons ont généralement des bordures pour les rendre visibles. Vous pouvez personnaliser l'épaisseur, la couleur, et le style des bordures.
+c. padding : Espacement interne
Le padding permet d'ajuster l'espacement interne entre le texte du bouton et ses bords.
+d. font-size : Taille du texte
Le font-size vous permet de modifier la taille du texte du bouton pour le rendre plus grand ou plus petit.
+e. border-radius : Coins arrondis
Les coins arrondis peuvent être obtenus en utilisant la propriété border-radius.
+f. cursor : Changer le curseur
Changer le curseur lors du survol du bouton améliore l'expérience utilisateur en indiquant clairement qu'il s'agit d'un élément cliquable.
3. Effets de survol (:hover) et de clic (:active)
Les effets visuels au survol et au clic aident à améliorer l'interaction utilisateur. Ces effets peuvent être réalisés avec les pseudo-classes :hover et :active.
+a. Effet de survol (:hover)
Lorsque l'utilisateur survole le bouton, vous pouvez modifier son apparence, par exemple, en changeant la couleur de fond ou en appliquant une ombre.
+b. Effet de clic (:active)
L'effet :active est activé lorsque l'utilisateur clique sur le bouton. Il peut être utilisé pour ajouter un effet de pression ou d'ombre.
